class 说明
.ui-col-xs- 加上 1-24 当分辨率大于 320px，都按比例占宽，否则宽度为 100%
.ui-col-sm- 加上 1-24 当分辨率大于 768px，都按比例占宽，否则宽度为 100%
.ui-col-md- 加上 1-24 当分辨率大于 992px，都按比例占宽，否则宽度为 100%
.ui-col-lg- 加上 1-24 当分辨率大于 1200px，都按比例占宽，否则宽度为 100%
.ui-col-xl- 加上 1-24 当分辨率大于 1600px，都按比例占宽，否则宽度为 100%
