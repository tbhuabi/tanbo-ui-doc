<h2 uiAnchor id="输入属性">输入属性</h2>

| 输入属性名     | 输入类型      | 默认值 | 说明    |
| --            | --                | --    |--     |
| `type`        | `line \| circle`  | line  |显示类型     |
| `value`       | `number`          | 1     |进度值     |
| `min`         | `number`          | 0     |进度最小值   |
| `max`         | `number`          | 100   |进度最大值   |
| `lineWidth`   | `number`          | 6     |进度条宽度   |
| `size`        | `number`          | 100   |当 type 为 circle 时，进度条的大小   |
